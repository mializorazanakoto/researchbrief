# Deep Space Research Brief: In-Space Manufacturing

This project contains a professional research brief on In-Space Manufacturing (ISM), developed as part of an AI Workshop challenge.

## Content
- **1-Page Brief**: A clean, academic-style presentation of recent literature (2023-2025).
- **Citations**: 5 high-impact papers with direct DOI links.
- **Design**: Responsive HTML/CSS layout optimized for reading and printing.

## Deployment to GitHub Pages

To deploy this brief to your own GitHub account:

1. **Initialize Git**:
   ```bash
   git init
   git add .
   git commit -m "Initialize deep space manufacturing brief"
   ```

2. **Create a Repository**:
   Create a new public repository on [GitHub](https://github.com/new) named `deep-space-brief`.

3. **Push to GitHub**:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/deep-space-brief.git
   git branch -M main
   git push -u origin main
   ```

4. **Enable Pages**:
   - Go to your repository **Settings** on GitHub.
   - Select **Pages** from the sidebar.
   - Under **Build and deployment**, ensure the source is set to "Deploy from a branch" and the branch is `main`.
   - Click **Save**. Your brief will be live at `https://YOUR_USERNAME.github.io/deep-space-brief/`.
